About the game

We have completed the preliminary project in first period. There is game target for users. The game provides different options to users. For example, the user can choose some European capital cities to visit. There is feedback information from the system. The information was stored in the new-created table. We think that we achieved our aim of original design.

Of course, there should be improvement and modifications in this game. We have discussed about the adjustments, meanwhile we also compare our project with the other groups projects. There are different targets in other groups. For instance: some projects would calculate the consumption of CO2, time zone and so on. We agree that there should be more quantity target in our project. We will add more data in further with usage of API. Just like we did in our assignment, we could use weather API, it provides instant information from most of the big cities all around the world.

In the second period, we will also learn HTML, CSS. We are thinking about how to combine the project with these new languages. But that is the preliminary idea. We did not decide how to complete it yet. In our opinion, it should be more complicated than the former one and provide more fantastic experience for users.
